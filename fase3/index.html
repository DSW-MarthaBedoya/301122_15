Bienvendio


Es este espacio encontraras, las enseñanzas para dinamizar el aprendizaje de las temáticas básicas de: Etiquetas Básicas de HTML, Atributos y Valores, los cuales permiten conformar una página web, con elementos primordiales tales como: párrafos, imágenes, viñetas, entre otros, a través de estas síntesis se conforma la página web de acuerdo con las necesidades y requerimientos del usuario.

Gracias al contenido de las etiquetas básicas de HTML (Lenguaje de Marcas de Hipertexto, del inglés - Hyper Text Markup Language)., se pueden desarrollar páginas y documentos web incorporando etiquetas que contienen información adicional ya sea de la estructura del texto o de su presentación.
Este es el lenguaje predominante en la construcción de páginas web, siendo hoy en día la Interfax más extendida en la red, donde la pieza clave es su “navegador” o “cliente” HTML, aquí quedan o reposan los registros de todas las codificaciones, que nos mas que instrucciones para el visualizador, es decir es el lenguaje usado para mostrar las paginas web a los usuarios.

=



Los atributos y valores son dos términos que van asociados, uno depende del otro, siendo una estructura básica de la codificación HTML, las etiquetas también denominadas tags, van cortejadas de Atributos, que siempre se deben encontrar en la etiqueta inicial, son palabras especiales ubicadas dentro de esta etiqueta. Ej. < a href = “http: // …..” >, y adicional se encuentran los Valores, que son aquellos que siempre debe permanecer encerrados en comillas, ya sean simples o dobles. ej. “imagen.png”.



Por lo anterior se dice que el Atributo y Valor deben estar unidos por el signo: =
